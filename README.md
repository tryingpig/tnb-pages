# TNB Pages

TNBfolio market-intelligence 리포트 아카이브 (GitHub Pages).

텔레그램 `TNB premium` 채널에서 올라온 HTML 리포트를
`telegram-forwarder` 의 `forwarder/publisher.py` 가 자동으로 이 레포에 게시한다.

- `index.html` — 미니멀 갤러리(자동 생성, 직접 편집 금지)
- `reports/` — 게시된 원본 HTML 리포트
- `manifest.json` — 리포트 목록 데이터(제목·날짜·파일)

공개 URL: https://tryingpig.github.io/tnb-pages/
